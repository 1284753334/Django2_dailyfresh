# 天天生鲜-django2.2版本

## 简介
**本项目替换原项目框架django1.8为最新版的django2.2.5**，该项目包含了实际开发中的电商项目中大部分的功能开发和知识点实践，是一个非常不错的django学习项目，同时也记录在替换框架中遇到的坑，希望对各位有所帮助。

## 开发环境  
```text
python:3.6.5
django:2.2.5
pycharm:2019.2
OS: win10
```
